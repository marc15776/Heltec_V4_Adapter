# Heltec LoRa V4 Adapter

This is a fork from [kiu/lofence](https://github.com/kiu/lofence)

I used the the resistor and opamp concept to use  with the [Heltec LoRa V4](https://heltec.org/project/wifi-lora-32-v4/) adc input.
I dont need the microcontroller, LoRa chipset and battery management since that is already integrated on the Heltec V4 board.
I will probably use this in point to point mode with another Heltec board.
